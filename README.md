# Global MT

Global MT is a Python-based multi-tool that provides a secure and efficient way to store and manage your passwords. It ensures privacy by storing passwords in a hidden file that cannot be accessed without proper authentication. The tool features a command prompt-style interface for easy interaction.

## Features
<details>
  <summary>Click to expand</summary>
  
  - Secure password storage in a hidden file
  - Authentication required to access stored passwords
  - Command-line interface for user-friendly interaction
  - Lightweight and easy to use
  
</details>

## Installation
<details>
  <summary>Click to expand</summary>
  
  1. Download the project files from the repository.
  2. Alternatively, download the ZIP file and extract it to your desired location.
  3. Install required dependencies:
     ```sh
     pip install playsound requests colorama
     ```
  
</details>

## Usage
<details>
  <summary>Click to expand</summary>
  
  1. Run the tool:
     ```sh
     python global_mt.py
     ```
  2. Follow the command prompt instructions to:
     - Store new passwords
     - Retrieve stored passwords (authentication required)
     - Manage existing passwords
  
</details>

## Security
<details>
  <summary>Click to expand</summary>
  
  - Passwords are stored in a hidden file, making them difficult to detect.
  - A master password is required to access stored passwords.
  - Uses encryption to enhance security.
  
</details>
